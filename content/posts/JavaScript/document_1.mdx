2024년 강원도에서는 교육 분야에서 다양한 이슈가 부각되었습니다. 주요 사안을
정리하면 다음과 같습니다:

학령인구 감소 현실화: 초등학교 신입생 수가 처음으로 1만 명 이하로
떨어졌으며, 16개 초등학교는 신입생이 한 명도 없었습니다. 또한, 6개 분교장이
통폐합되었습니다. 어린이집 원생 수는 6년 만에 30% 감소하였고, 연평균 70개
어린이집이 문을 닫았습니다. 강원일보

의대 정원 확대에 따른 갈등: 정부 정책으로 도내 4개 의대 모집 정원이
267명에서 391명으로 늘었고, 지역인재 전형으로 146명 선발이 확정되었습니다.
그러나 이 과정에서 교수진 집단 사직과 학생 동맹휴학 등 갈등이
발생하였습니다. 강원일보

교육당국과 교사단체 간 갈등 심화: 강원교육청과 전교조 강원지부 간의 갈등이
심화되었습니다. 도교육청이 전교조와의 단체협약 실효를 선언하면서 반발이
커졌으며, 신경호 교육감이 학교 현장 방문 중 전교조와의 마찰로 실신하는 일도
발생했습니다. 강원일보

국립대 통합 논의: '강원 1도 1국립대'를 목표로 한 통합 논의가
본격화되었습니다. 강원대와 춘천교대 통합이 주요 안건으로 떠오르며,
강원도립대 합류 가능성도 제기되었습니다. 그러나 예산 문제로 난항을 겪기도
했습니다. 강원일보

예산 부족으로 인한 교육환경 위기: 세수 부족으로 강원교육 예산이 큰 타격을
입었습니다. 올해 교육교부금이 2,407억 원 부족하여 교육기금에서 4,000억여
원을 충당해야 했으며, 내년에도 5,800억 원의 교육기금이 추가로 필요할
전망입니다. 강원일보

이러한 이슈들은 강원도 교육계에 큰 영향을 미쳤으며, 지역 사회에서도 많은
논의가 이루어졌습니다.

```tsx
import { CloudImage } from "@/components/cloud-image";
import { Darkmode } from "@/components/darkmode-switch";

import { posts } from "../../.velite";

import Link from "next/link";

const sortedByDatePosts = posts.sort((a, b) => {
  if (a.date > b.date) return -1;
  if (a.date < b.date) return 1;

  return 0;
});

const Header = () => {
  return (
    <header className="relative flex flex-col items-center gap-8 md:w-1/3">
      <CloudImage />
      <Darkmode />
    </header>
  );
};

const PostsList = () => {
  return (
    <section className="scrollbar-hide w-full md:h-[85vh] md:max-h-screen md:w-1/2 md:overflow-y-scroll">
      <ul className="space-y-8">
        {sortedByDatePosts.map(({ title, slug, date }) => {
          return (
            <li
              key={slug}
              className="rounded-lg border-[1px] border-neutral-500/20 bg-neutral-100 dark:border-white/20 dark:bg-[#303030]"
            >
              <Link
                href={slug.toLowerCase()}
                className="block h-full w-full space-y-4 p-4 md:p-6"
              >
                <p className="text-lg font-[400]">{title}</p>
                <p className="flex justify-between text-xs">
                  <span>{slug.split("/")[1]}</span>
                  <span>{date}</span>
                </p>
              </Link>
            </li>
          );
        })}
      </ul>
    </section>
  );
};

export default function MainPage() {
  return (
    <div className="max-h-screen space-y-8 px-4 pb-12 pt-4 md:flex md:max-w-[1024px] md:items-center md:justify-between">
      <Header />

      <PostsList />
    </div>
  );
}
```
